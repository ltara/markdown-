## 粗体和斜体

### 粗体
一盏**灯**，一片昏黄；一简书，一杯淡茶。守着那一份淡定，品读属于自己的寂寞。**保持淡定**，才能欣赏到最**美丽的风景**！保持淡定，人生从此不再寂寞。

### 斜体
一盏灯，一片*昏黄*；一简书，一杯*淡茶*。守着那一份淡定，品读属于自己的*寂寞*。保持淡定，才能欣赏到最美丽的风景！保持*淡定*，人生从此不再寂寞。
